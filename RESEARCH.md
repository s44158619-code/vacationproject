# 아힘하 사이트 설득 구조 메모

## 조사한 유사 방향

- 아이템스카우트: 스마트스토어와 이커머스 셀러를 위한 키워드 분석, 상품 발굴, 랭킹 추적을 강조한다.
- 판다랭크: 크리에이터와 셀러를 위한 키워드 분석, 상품 분석, AI 자동화, 수익화 도구를 강조한다.
- 셀메이트: 쇼핑몰 통합관리, 재고관리, 물류, WMS처럼 규모가 있는 판매 운영을 강조한다.
- 사방넷: 여러 쇼핑몰 주문, 상품, 재고, 배송을 하나로 관리하는 통합관리 성격이 강하다.
- 샵링커: 상품관리, 주문관리, CS/클레임, 재고관리, 송장출력, 교육, 고객지원, 요금 안내를 촘촘하게 배치한다.
- 플레이오토: 쇼핑몰 연동, 택배사 연동, 요금 비교, 후기, 업종별 사용 사례, 회사/약관/개인정보 링크를 갖춘다.

## 아힘하의 차별점

큰 도구들은 데이터 분석이나 대형 쇼핑몰 통합관리에 강하다. 반면 초보 공구 판매자는 그 전에 주문 누락, 입금 확인, 마진 계산, 고객 안내문, 상세페이지 문구에서 먼저 막힌다.

그래서 아힘하는 다음 메시지를 중심으로 잡는 것이 좋다.

- "공구 판매자가 오늘 당장 쓰는 운영 도구"
- "주문표, 마진표, 안내문, 상세페이지 문구를 한 번에 정리"
- "큰 솔루션이 부담스러운 초기 판매자를 위한 작고 실용적인 시작점"

## 사이트에 반영한 설득 흐름

1. 첫 화면은 긴 설명보다 이미지 슬라이드와 한 문장 가치 제안으로 구성했다.
2. 아래 섹션에서 공구 판매자가 겪는 문제를 주문, 마진, 반복 문의로 분리했다.
3. 유사 서비스와 비교해 아힘하의 위치를 "초기 공구 운영 실무"로 좁혔다.
4. 서비스 카드는 주문관리표, 상세페이지 문구, 운영 문구 패키지로 단순화했다.
5. 로그인, 회원가입, ID/PW 찾기, 고객센터, 장바구니, 공지사항은 실제 기능 연결 전이라도 사이트 신뢰감을 주도록 팝업 형태로 만들었다.
6. 인스타그램과 카카오톡은 링크 자리를 비워두어 나중에 계정 정보를 받으면 바로 연결할 수 있게 했다.

## 2차 개선에서 반영한 것

- 공지사항, 이용가이드, 고객센터, 장바구니를 팝업이 아니라 실제 하위 페이지로 분리했다.
- 서비스 페이지에서 장바구니에 실제로 담고, 수량 변경, 삭제, 합계 계산이 되도록 만들었다.
- 결제 준비 페이지를 추가해 주문자 정보와 요청사항을 입력하는 흐름을 만들었다.
- 마이페이지를 추가해 향후 주문 내역, 상담 상태, 제작 파일을 보여줄 자리를 마련했다.
- 이용약관과 개인정보처리방침 페이지를 추가해 실제 결제 연결 전 신뢰 구조를 준비했다.
- 메인에 조사 기반 섹션을 추가해 아힘하가 대형 솔루션과 다르게 "초기 공구 운영"에 집중한다는 메시지를 강화했다.

## 레퍼런스에서 배운 기본 구성

- 데이터/키워드형 사이트: 무료 시작, 키워드 분석, 상품 분석, 순위 추적, 가이드, FAQ가 중요하다.
- 통합관리형 사이트: 상품관리, 주문관리, 재고관리, CS/클레임, 배송/송장, 요금 안내, 고객지원, 보안/신뢰 요소가 중요하다.
- 아힘하에 필요한 구성: 대형 기능보다 먼저 주문표, 마진표, 안내문, 장바구니, 상담 접수, 공지, 가이드, 약관 자리를 갖춰야 한다.

## 가격검색 기능 조사

- KAMIS는 농수축산물 가격정보와 Open API를 제공한다. 가격정보 메뉴에는 소매가격, 도매가격, 중도매인 판매가격, 친환경농산물 소매가격 등이 있다.
- 농넷은 농산물 유통 종합정보 시스템으로, 향후 가격/유통 데이터 확장 참고처로 적합하다.
- 실시간 가격은 품목, 등급, 지역, 조사일에 따라 크게 달라지므로 사이트에 고정값만 넣으면 위험하다. 따라서 1차 구현은 기준가 샘플과 직접 수정 계산기로 만들고, 추후 KAMIS API 키를 받아 실시간 연동하는 구조가 적합하다.
- 식당 메뉴 마진은 단일 상품 판매와 달리 여러 재료의 g 단위 사용량을 합산해야 한다. 그래서 kg 단가를 1g 단가로 환산하는 레시피 계산기를 추가했다.
- 쿠팡, 지마켓, 네이버 쇼핑 가격은 공식 제휴/API 또는 사용 허가 없이 자동 수집하면 안정성과 약관 문제가 생길 수 있다. 그래서 현재는 마켓별 기준가 샘플을 보여주고, 각 마켓 검색 URL로 이동하는 버튼을 제공한다.
